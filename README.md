# Приложение для прогноза погоды

Веб‑приложение на чистом HTML/CSS/JavaScript, которое показывает текущую погоду и прогноз, умеет работать с геолокацией, поиском по городу и хранит выбранные города между перезагрузками. 
## Что сделано в работе

- При первом открытии приложение запрашивает геолокацию и автоматически показывает погоду для текущего местоположения (в интерфейсе выводится как «Текущее местоположение»). 
- Если геолокация недоступна/отклонена, пользователь может вручную ввести город и получить погоду и прогноз. 
- Реализован основной поиск города с выпадающими **подсказками** при вводе (часть подсказок берётся из захардкоженного списка, часть — через geocoding API). 
- Добавлена возможность указать ещё **2 дополнительных города** и получать погоду для них в отдельных блоках. 
- Кнопка «Обновить» делает повторные запросы к API без перезагрузки страницы. 
- Реализованы состояния интерфейса:
  - лоадер во время запросов;
  - обработка ошибок (например, если город не найден или API недоступно). 
- Сохранение состояния в `localStorage`:
  - последний основной город;
  - два дополнительных города;
  - список недавних городов (для подсказок). 
- Адаптивная вёрстка: приложение корректно отображается на десктопе и на мобильных устройствах, без наложения элементов. 

## Используемые технологии

- HTML5, CSS3 (без фреймворков) 
- JavaScript (без фреймворков)
- OpenWeatherMap API: текущая погода, прогноз и геокодинг (поиск городов).
